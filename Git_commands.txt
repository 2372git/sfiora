cd - проверять, что находимся в папке локального депозитория
git init - превращает в репозиторий текущую папку, в которой находимся.
git remote add origin {url} - связывает свой локальный депозиторий с удаленным

git clone {url} - склонировать удаленный депозиторий в свою локальную папку

Коммит - сохраненная текущая редакция файла (файлов) в репозитории

git status - состояние локального деп (файлы, которые изменились, которые не добавлены в отслеживание или если ничего интересного)
git add {файлы, папки или .} - добавление файлов, папок или всего в депозитории в отслеживание гита
git commit -m "комментарий" - создание коммита из отслеживаемых файлов (в коммит попадут только измененные из добавденных git add) с комментарием-напоминалкой
git log - просмотр всех комитов
